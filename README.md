# Quick Commerce SQL Project

## Project Overview
This project is an end-to-end SQL analysis on a Kaggle quick commerce dataset.
The goal is to analyze customer behavior, orders, transactions, ratings,
and delivery performance using MySQL.

## Dataset
Source: Kaggle

Tables used:
- customers
- products
- orders
- transactions
- ratings
- delivery

## Objectives
- Understand customer demographics
- Analyze order and transaction patterns
- Evaluate delivery performance
- Study customer ratings and satisfaction

## Tools Used
- MySQL
- GitHub
